# Component Library Demo

This repository contains **accessible web UI components** with live demos and source code.  
Each component is self-contained with HTML, CSS, and JS for easy learning and reuse.

---

## Components

| Component      | Description                                    | Demo Link                          |
|----------------|-----------------------------------------------|-----------------------------------|
| Autocomplete   | Accessible autocomplete input with filtering and “No results” messaging | [Live Demo](components/autocomplete/index.html) |
| Accordion      | Expandable/collapsible sections with keyboard and screen reader support | [Live Demo](components/accordion/index.html) |
| Modal          | Accessible modal dialog with focus trapping   | [Live Demo](components/modal/index.html) |
